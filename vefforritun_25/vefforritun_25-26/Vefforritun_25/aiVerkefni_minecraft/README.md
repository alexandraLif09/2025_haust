# MicroCraft — Frontend-only Minecraft-like demo


Skrár: index.html, styles.css, js/*


Hvernig á að keyra: opna `index.html` í nútímalegum vafra (Chrome/Edge/Firefox). Nota `http` er betra (sumir vafrar takmarka pointer lock þegar stofnað frá file://).


Eiginleikar:
- ES Modules, split-in files
- Chunked seeded terrain
- Lightweight controls (WASD + pointer lock)
- Day-night cycle and dynamic sky color
- Hotbar UI and minimap placeholder


Breyta/útfæra: bæta við textures, meshing optimisation, block interactions (place/remove) og fleiri block-a.
